# Screen Mockups

Overview of all screens in Stagr. Links open in the browser.

## Player screens (phone)

- [Lobby / Join](lobby.html) — Scan QR code, receive character, wait for game to start
- [Main screen](player-main.html) — Character, current task, tap to roll, mood, roll history
- [Standup](player-standup.html) — Pick a task, choose to collaborate
- [Waiting (after roll)](player-waiting.html) — Mood flavour text, prompt to look at the shared screen
- [Character details](player-character-details.html) — Full dot grid per skill, happiness history
- [Hiring — candidate view](player-candidate.html) — Own dot grid visible during the interview

## Facilitator screens (large display)

- [Dashboard](facilitator-dashboard.html) — Kanban board, CFD, cycle time scatter plot
- [Hiring overlay](facilitator-hiring.html) — QR code for candidates, team skill coverage
